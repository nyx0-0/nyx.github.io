HOW TO USE BACON STUDY APP

STUDY MODE

  • Enter a number, and then click the button of the desired base representation.
  • The box on the right will display steps on how to transform the number to its base form.

FLASHCARD MODE

  • Set the timer (in seconds). This represents how long a question will be flashed on the screen before moving onto the next.
  • Set the number of questions to be generated.
  • Click the "Next Card" button to move onto the next question without having to wait for the timer to end.
  • All of the correct answers will be shown after all the questions have been flashed on the screen.

QUIZ MODE

  • Set the number of questions to be generated.
  • Click the "START" button and a number (integer) will be flashed ont he screen together with the base representation that you will have to convert it to.
  • Type the answer on the text box and click "SUBMIT" to move forward to the next question.
  • There is a "SKIP" option, but clicking it would automatically mean that the question will not be scored.
  • The score will be shown after all the questions have been flashed on the screen.

CALCULATOR

  • The fourth page is a Base Converter with an integrated calculator.
  • Click first the desired base representation (Base 2, Base 8, or Base 16) followed by the base value desired (e.g. 1001001, 42, 2A).
  • The user can make normal calculations with decimal representations (whole numbers only), as well as base representations.
